## Eight Queens Problem

This problem was solved using GA.

**Objective Function**

$MinZ = Iteration$

**Components of Algorithm**

+ Roulette Selection
+ Permutative Crossover
+ Permutative Mutation

**Parameters of Algorithm**

+ Population Size
+ Crossover Rate
+ Mutation Rate
